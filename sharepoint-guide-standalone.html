SharePoint Permissions Guide

Step-by-step instructions for managing site-level and file/folder-level permissions in SharePoint.

🏢

Site permissions control who can access your entire SharePoint site. Users inherit these permissions across all files and folders unless you set specific permissions on individual items.

In the top-right corner of your SharePoint site, click the ⚙️ Settings icon. A dropdown menu will appear.

From the Settings dropdown, select Site permissions. This opens the Site Permissions panel on the right side.

At the bottom of the Site Permissions panel, click Advanced permissions settings. This takes you to the classic SharePoint permissions page.

💡 The advanced settings page gives you granular control over who has access and at what level.

On this page, you'll see three default SharePoint groups. Each group has a different permission level:

SharePoint Groups

GroupPermission Level

Click on one of the group names (e.g., Site Members) to see all users in that group.

Inside the group, tick the checkbox next to the user(s) you want to remove. Then, click Actions → Remove Users from Group in the ribbon at the top.

 Actions → Remove Users from Group
⚠️Removing a user from a site group removes their access to the entire site and all its contents (unless they have specific permissions on individual files/folders).

✅ Result

The user is removed from the group and loses the associated permissions across the entire SharePoint site.

Go to the Document Library (or folder) where your file or folder is located. You can access this from the left navigation or via Site contents.

💡 Make sure you're viewing the library in list or grid view where you can see all items.

Click the checkbox next to the file or folder name to select it. Don't open it — just select it so the row highlights.

Hover over the selected item and click the three dots (⋯) that appear to open the context menu.

💡 You can also right-click the file or folder to open the same context menu.

In the context menu, find and click Manage access. This opens the Manage Access panel on the right side of the screen.
